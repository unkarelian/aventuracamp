Getting Started

Setup Guide

Get Aventura running in minutes. Choose your AI provider and start creating stories.

1

Download Aventura

Download the latest version of Aventura for your platform. The app is available for Windows, macOS, Linux, and Android.

desktop_windows Windows laptop_mac macOS terminal Linux android Android
2

Choose Your AI Provider

When you first launch Aventura, you'll be asked to choose an AI provider. Each option has different benefits depending on your needs.

OpenRouter

Access multiple AI models through a single API. Supports 70+ providers including OpenAI, Anthropic, Google, and more.

  • check_circle 70+ model providers
  • check_circle Single API key
  • check_circle Pay-per-use pricing
  • check_circle No vendor lock-in
Get API Key code

NanoGPT

Affordable AI API with access to models like Deepseek and GLM. Great for budget-conscious users.

  • check_circle Affordable pricing
  • check_circle Deepseek models
  • check_circle Thinking model support
  • check_circle Simple setup

For thinking models, append :thinking to the model name (e.g., deepseek/deepseek-v3.2:thinking)

Get API Key code

Custom / Self-Hosted

Configure your own OpenAI-compatible API endpoint. Perfect for self-hosted models or enterprise deployments.

  • check_circle Full control
  • check_circle Privacy-focused
  • check_circle Use any compatible API
  • check_circle No external dependencies
